# About 
##This project is a web-based sales visualization tool designed to analyze the weekly, per-day sales performance of the food product Pink Morsel.

Key features include:
1. Interactive visualizations of daily sales trends.
2. Region-wise analysis to compare performance across different markets.
3. Highlighting sales rate changes before and after price adjustments.
4. Data-driven insights to support strategic decision-making for product pricing and regional targeting.

## Tech Stack
- Data_Handling: pandas
- Visualization: Dash(python framework)
